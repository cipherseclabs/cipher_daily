Weekly Brief – August 19–25, 2025

Cybersecurity Events & Incidents

  • A new Chinese state‑linked group, UAT‑7237 (likely a subgroup of UAT‑5918), has been observed targeting web‑hosting companies in Taiwan. Techniques include exploiting unpatched public servers for access, deploying a custom shellcode loader “SoundBill,” using Cobalt Strike, RDP, and SoftEther VPN for persistence and credential theft. Source: https://www.techradar.com/pro/security/chinese-hackers-are-targeting-web-hosting-firms-heres-what-we-know
  • Russian government‑linked actors (FSB’s Center 16, aka Static Tundra/Berserk Bear) are exploiting the old Cisco Smart Install (CVE‑2018‑0171) in unpatched network devices across sectors (telecom, education, manufacturing) globally. They leverage SMI and SNMP for stealthy configuration modifications and long‑term spying. Source: https://www.itpro.com/infrastructure/networking/russian-hackers-are-using-an-old-cisco-flaw-to-target-network-devices-heres-how-you-can-stay-safe
  • Aspire Rural Health System (Michigan) disclosed a ransomware breach affecting 138,386 individuals—medical and financial data exposed, attributed to the BianLian ransomware group. Source: https://social.cyware.com/category/breaches-and-incidents-news
  • A data breach exposed nearly 1 million U.S. medical cannabis patient records (Social Security numbers, government IDs, health data); the dataset was taken offline after the company was alerted. Source: https://www.wired.com/story/us-government-seeks-medical-records-of-trans-youth

Vendor & Security Product Updates

  • Google released emergency Chrome updates to address high‑severity vulnerabilities (e.g., CVE‑2025‑8901 in the ANGLE graphics engine). Users are urged to update and restart immediately. Source: https://www.wired.com/story/us-government-seeks-medical-records-of-trans-youth
  • Cisco patched critical CVE‑2025‑20265 (10/10 severity) in Secure Firewall Management Center’s RADIUS subsystem—vulnerable to remote arbitrary shell‑command injection. Patch or disable RADIUS auth recommended. Source: https://www.techradar.com/pro/security/cisco-warns-of-worrying-major-security-flaw-in-firewall-command-center-so-patch-now
  • Microsoft’s August updates (KB5063878 for Windows 11 24H2; KB5063709 for Windows 10/11) are causing streaming app performance issues (OBS, NDI Tools), particularly affecting RUDP. Workaround: switch to TCP/UDP NDI mode or uninstall updates. Source: https://www.tomsguide.com/computing/windows-operating-systems/microsoft-confirms-latest-windows-11-update-is-tanking-streaming-app-performance
  • Another Microsoft August update (KB5063875) caused SSD data loss and broke reset/recovery functions on Windows 10 and early Windows 11. Microsoft released an out‑of‑band fix and advised using the corrected update. Source: https://www.itpro.com/software/windows/microsofts-botched-august-update-batch-first-wiped-ssds-now-its-breaking-pc-resets-and-recoveries-on-windows

Analyst Comments / Defender Impact Summary

Analyst Comments: Chinese and Russian state‑linked actors continue to employ stealthy, infrastructure‑level access routes into critical systems—pursuing long‑term espionage, not rapid disruption. Microsoft and Cisco patches the same week underscore the persistent need to prioritize remediation, as failures or performance issues can cascade into operational disruption.

Defender Impact Summary: Organizations should immediately audit exposure to Cisco Smart Install and apply patch or disable, deploy updated FMC RADIUS fixes, patch Microsoft August vulnerabilities (especially CVE‑2025‑53779), and monitor for BianLian and unknown infrastructure-based intrusions. Updates from major vendors carry regression risk—ensure rollback plans exist.
